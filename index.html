Vite Plugin · Open Source

Click Element
→ Claude Code → Done

在 Vite 项目中 Shift+Alt 点击任意元素，
弹出 Claude Code 面板，描述你想改什么，代码自动修改、页面自动刷新。
只需 ccPromptPlugin() 一行配置。

Get Started → GitHub

1. Install

npm install cc-prompter

2. Configure

// vite.config.ts import { ccPromptPlugin } from 'cc-prompter'; export default defineConfig({ plugins: [ ccPromptPlugin(), // ← 就这一行 react(), ], });

3. Done

npm run dev # Shift+Alt 点击页面元素 → 面板弹出 → 输入修改指令 → 自动完成

How It Works

1

Shift + Alt 悬停元素

code-inspector 在编译时为每个 DOM 元素注入源码定位信息。按住 Shift + Alt 悬停，自动读取 data-insp-path 属性，获取文件路径、行列号。

2

点击 → 面板弹出

点击后弹出一个可拖拽、可缩放的浮动 iframe 面板。源码定位信息自动附加到当前活跃 session，输入框自动聚焦。

3

描述修改 → Claude 执行

输入「把按钮改成圆角蓝色」，Claude Code 通过 PTY 直接操作你的代码文件。工具调用（Read/Edit/Write）按序实时展示。

4

HMR 自动刷新

代码修改保存后，Vite HMR 自动热更新页面，无需手动刷新。你可以继续下一次操作。

Features

🎯

元素级源码定位

内置 code-inspector-plugin，编译时打标、运行时交互。点击即获取精确的文件路径、行列号和 DOM 元素信息。

💬

多 Session 并行

多个独立 Claude 会话同时运行，各有自己的上下文和源码定位。Tab 切换互不干扰，支持同时在不同 session 中操作。

⚡

实时流式渲染

Markdown 实时渲染（标题、代码块、列表、加粗/斜体），工具调用按序展示，带进度指示和打字动画光标。

⏹

随时中断

生成过程中点击 Stop 按钮，发送 Escape 信号中断 Claude。会话不会销毁，可以继续对话。

🖼

灵活面板

浮动 iframe 面板，支持拖拽移动、边缘和四角缩放。Ctrl+Shift+P 随时召回，Esc 隐藏。

🔌

零配置集成

一个插件包含 code-inspector + PTY sidecar + 脚本注入。端口自动选择，dev-only 注入，生产构建零侵入。
